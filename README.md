# livecode4.21-N-1

### Exo A
   À l'aide du module turtle, écrivez un script qui dessine 4 étoiles dans une fenêtre avec des lignes de differente couleurs
   - Les lignes de l'étoile doivent avoir des couleurs différentes.

### Exo B
  
  Améliorez le projet le nombre magique de sorte à ce que l'utilisateur puisse effectuer une mise.

  - L'utilisateur possède 200 pièces au départ
  - La mise est de minimum 50 pièces
  - A chaque échec le nombre de pièces misé lui est retranché
  - A chaque victoire, il obtient le triple de sa mise
  - Pensez à vérifier si l'utilisateur possède la somme qu'il souhaite miser
  + on négligera la limitation du nombre de vies

NB:
  L'Étudiant devra choisir l'un des exercices à traiter.
